


### create a container

```bash
docker pull nagabhushanamn/java-web-service:batch9
docker create -d -p 8080:8080 --name java-web-service nagabhushanamn/java-web-service:batch9
docker start java-web-service
# or
docker run -d -p 8080:8080 --name java-web-service nagabhushanamn/java-web-service:batch9
```
